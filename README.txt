Ведется разработка для отслеживания запущенных программ на клиентских компьютерах.
Необходимо написать программу, которая будет отслеживать(путем обновления) запущен-
ные приложения и отсылать на сервер, где вся эта информация будет вноститься в БД.

Необходимая информация от клиента:
*ID
*Время обновления
*Cписок приложений, где:
			**ID
			**Имя приложения
			**Когда приложение было запущено.
			**Предположительно время завершения приложения
